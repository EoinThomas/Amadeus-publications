# Diverse and Private Synthetic Datasets Generation for RAG Evaluation: A Multi-Agent Framework

[← Back to Publications](/Amadeus-publications/)

- **Venue:** TRUST-AI @ ECAI 2025
- **Year:** 2025
- **Link:** [arXiv](https://arxiv.org/abs/2508.18929)

## Abstract

This paper proposes a multi-agent system to generate synthetic datasets for evaluating Retrieval-Augmented Generation (RAG) models, ensuring dataset diversity and privacy. The framework combines multiple generative agents to produce data that mimic real-world distributions without exposing sensitive information, enabling robust assessment of RAG performance.

## Key Contributions

- Multi-agent framework for synthetic dataset generation
- Privacy-preserving data synthesis for RAG systems
- Diversity-focused generation strategies
- Evaluation methodology for RAG applications
