# Semantic Adapter for Universal Text Embeddings: Diagnosing and Mitigating Negation Blindness to Enhance Universality

[← Back to Publications](/Amadeus-publications/)

## Publication Details

- **Authors:** Cao, H.
- **Venue:** ECAI 2025
- **Year:** 2025
- **Link:** [arXiv](https://arxiv.org/abs/2504.00584)

## Abstract

This work identifies and addresses the issue that certain universal text embedding models fail to handle negation, proposing an adapter to improve embedding robustness regarding negated meaning. The semantic adapter enhances universality of text embeddings by diagnosing and mitigating negation blindness, a common limitation in embedding models.

## Key Contributions

- Diagnosis of negation blindness in universal text embeddings
- Novel semantic adapter architecture
- Improved handling of negation and semantic nuances
- Enhanced universality across diverse tasks
